# plugin-photo-swiper

一款基于 vue+ typescript+cli3 的图片预览组件，可以进行左右切换图片预览，更好的是它还可以从点击元素进行无缝过渡，使得图片预览看起来个更加平滑，组件预览和轮播滚动他都做到了。 组件基于 photoswiper,所以兼容了他所有的可配置项，并进行了一些功能的改造。

## 演示地址

https://starfishing.github.io/plugin-site/

## 特性

- 移动端双指缩放，双击放大，左右滑动切换
- pc 端支持鼠标左右滑动切换，鼠标单击放大
- 支持图片的缩放归位效果，也支持单纯的图片预览
- 支持鼠标滚轮滚动关闭
- ...
