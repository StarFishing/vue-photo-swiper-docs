# 基本使用

现在在 Vue 模板文件内就可以使用 photo-swiper 标签了，我们添加一些必要的数据让组件运行起来

# 模板文件

<template>
  <div class="wrapper">
      <button @click="showPreview">打开图片预览</button>
      <photo-swiper  @close="handleClose"
                    :visible="visible"
                    :images="images"
      ></photo-swiper>
    </div>
  </div>
</template>

# js 文件

<script>
  export default {
    data(){
      return {
        visible:false,
        images:[
          {
            src:'lovely.png',
            w:100,
            h:100
          },
          {
            src:'boy.png',
            w:220,
            h:400
          }
        ]
      }
    },
    methods:{
      showPreview(){
        this.visible=true
      },
      handleClose(){
        this.visible=false
      }
    }
  }
</script>

现在点击按钮就可以看到图片预览了，可以把代码复制到本地进行测试。
