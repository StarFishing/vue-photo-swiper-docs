# 安装

plugin-photo-swiper 的安装非常简单

# 安装插件

npm install plugin-photo-swipe -s

# 在入口文件引入插件

import Vue from 'vue'
import 'plugin-photo-swiper/lib/photo-swiper.css'
import PhotoSwiper from 'plugin-photo-swiper'
Vue.use(PhotoSwiper)

css 文件是必要的，否则会出现无法显示的问题，这也是该插件默认提供的样式。可以在自己组件内覆盖原有样式，以达到自定义的效果
