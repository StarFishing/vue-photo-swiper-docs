# API

prop 类型 说明 必须 默认值
visible Boolean 是否打开图片预览，请在关闭后在回调中设置为 false，否则可能无法再次打开 是 false
options Object 插件相关的配置项，详情见 options 否 {}
images Array 图片数组，在不需要缩放归位的效果时传入，以便获得预览图片列表 否 []
gallerySelector String 图片展示的父容器,在需要缩放归位的效果时传入，以便自动获得内部元素的 src 否 ''
gallerySelectorItem String 图片的直接父元素,用于过渡效果的定位，在 gallerySelector 开启时开启 否 ''
openArrow Boolean 是否开启左右切换的箭头 否 false
pageIndex Object|Boolean 页数展示的位置，为 false 时不展示，也可以接收一个对象{position:string}可选值：center,left,right,默认为 left 否 false
fullscreen Boolean 是否开启全屏按钮，开启后 pc 端可进行全屏预览 否 false
zoom Boolean 是否在 pc 端开启放大按钮，开启后在可以点击放大按钮进行图片的放大，默认不开启 否 false

# 事件

名称 描述 参数
close 在预览关闭时触发 无

其他事件可能会陆续补充

名称 描述 可用参数
footerBar 底部插槽，用于自定义底部内容，默认高度 44px 无
topbar 顶部插槽，用于自定义顶部擦作栏 无
